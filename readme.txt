== Description ==
The Plugin automatically compress all your images. make your website faster by optimizing JPEG images. 
Just Active the Plugin your all JPEG Images automatically compress and also your images automatically optimized when upload in folder.


== Features ==
- Optimize & compress images
- FREE Plugin.
- One click to Install.
- Simple and Easy to Use
- Quick and supportive.
 
== Installation ==
1. Install as a regular WordPress plugin from plugin library or copy and paste this files within your wp-plugins folder.
2. Activate the plugin through the 'Plugins' menu in WordPress admin.
